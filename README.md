# Rewards Tracker

This project was created to meet two primary goals. An employee of Oregon Tech
desired a simple application for tracking purchases/rewards. I desired an
experiment to evaluate modern AI tools in software development. My goal was to
understand how effectively AI could generate a functional desktop application
from requirements while I guided the design, validated the implementation, and
tested functionality.

## Objectives

- Evaluate AI's (Claude) ability to generate production-quality C# and WPF code
- Learn best practices for AI-assisted software development
- Plan/Design with AI (ChatGPT/Claude) based off requirements from the customer
- Understand and validate the generated architecture and business logic
- Practice debugging, testing, and refining AI-generated code

## Technologies

- C#/Xaml
- WPF
- .NET 8.0
- Git
- Microsoft Visual Studio Enterprise 2022
- AI-assisted development (ChatGPT, Claude)

## Features

- Track customer purchases, available rewards, and reward progress
- Customer management interface for names and phone numbers
- Persistent and portable data storage via OneDrive (customer's preferred choice)

## Outcomes of the project

This project demonstrated that AI can significantly accelerate development, but
successful software still requires human oversight and testing. Throughout
development I reviewed generated code, tested functionality, corrected issues,
and refined requirements. The simple, yet portable, application met the
customer's needs.
